# club 麒麟 Kirin Website

岡山・柳町の `club 麒麟 Kirin` 向け静的LPです。

## ローカル確認

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173/` を開きます。

## 公開前チェック

- `index.html`、`robots.txt`、`sitemap.xml` のURLが `https://clubkirin.com/` になっていることを確認
- 電話番号、住所、営業時間、定休日を店舗確認
- LINE、Instagram、TikTok、求人応募先URLを設定
- キャスト情報と写真の公開許諾を確認
