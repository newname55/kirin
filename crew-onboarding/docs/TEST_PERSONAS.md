# crew-onboarding テストペルソナ一覧

作成日: 2026-06-26
目的: 問診フローを人間目線でテストする（自動化前の手動テストケース）

---

## 使い方

1. ブラウザでチャットを開く
2. 以下のペルソナになりきって問診を進める
3. 「想定回答例」を参考に返答する
4. 問診完了後、管理画面で `candidate_score` / `priority_grade` を確認する
5. 「LINE応募傾向」と実際のCTA挙動が一致するか確認する

### P01 — あかり（未経験・ポジティブ）

**プロフィール**
- 年齢イメージ: 21歳
- 経験: なし（キャバクラ未経験。居酒屋バイト経験あり）
- 希望出勤: 週3〜4日
- お酒: 少し飲める
- 呼客: 体験日ならできそう（友人に声をかけてみる）

**不安ポイント**
- 「未経験でも本当に大丈夫?」
- 「最初の時給はいくら?」
- 「身バレしないか」

**LINE応募傾向:** する（前向きだが背中を押す一言が必要）

**想定回答例**
```
Q: キャバクラのお仕事経験はありますか?
A: ないです。でも興味はずっとあって。居酒屋は2年やってました。

Q: お酒は飲めますか?
A: ちょっとは飲めます。強くはないですけど

Q: 体験日に友人を誘えそうですか?
A: 頑張れば声かけられると思います。保証はできないですけど笑

Q: 出勤はどのくらい希望ですか?
A: 週3か4日くらいがいいです。Wワークなので
```

**確認観点**
- 未経験ルートで score が計算されるか
- 「身バレ不安」に対して適切な返答が出るか
- CTA が自然なタイミングで出るか

### P02 — みく（未経験・不安強め）

**プロフィール**
- 年齢イメージ: 19歳
- 経験: なし（初めて求人を見た）
- 希望出勤: 週1〜2日（お試しで）
- お酒: 飲めない
- 呼客: 難しい

**不安ポイント**
- 「本当にお酒飲めなくて大丈夫?」
- 「怖いお客さんが来たらどうするの?」
- 「最低いくらもらえる?」
- 「親にバレたら困る」

**LINE応募傾向:** しない（不安が解消されないと離脱する）

**想定回答例**
```
Q: キャバクラのお仕事経験はありますか?
A: ないです。求人見て初めて興味持ちました

Q: お酒は飲めますか?
A: 全然飲めないんですけど…大丈夫ですか?

Q: 体験日に友人を誘えそうですか?
A: ちょっと難しいです…

Q: 出勤はどのくらい希望ですか?
A: まずは週1か2日でお試ししたいです
```

**確認観点**
- お酒「飲めない」への返答が硬直しないか
- 不安系の質問（怖いお客・バレ）を入力したとき intent が正しく分類されるか
- LINE CTAが無理に出ないか（不安が解消されていない段階）

### P03 — さつき（経験者・即戦力）

**プロフィール**
- 年齢イメージ: 24歳
- 経験: あり（3年。前のお店は閉店）
- 希望出勤: 週5以上（メイン収入）
- お酒: 飲める
- 呼客: 週6組以上（固定客が複数いる）

**不安ポイント**
- 「時給いくら? 前のお店より低かったら難しい」
- 「指名料の取り分は?」
- 「体験入店の条件は?」

**LINE応募傾向:** する（条件確認後、即決タイプ）

**想定回答例**
```
Q: キャバクラのお仕事経験はありますか?
A: あります。3年やってました。前のお店が閉店して

Q: 前のお店での1ヶ月の指名本数や売上イメージはどれくらいでしたか?
A: 指名は月40〜50本くらいでした。売上は月100万前後

Q: 今すぐ連れてこられるお客様はいますか?
A: はい、7〜8組はすぐ声かけられます

Q: 出勤はどのくらい希望ですか?
A: 週5〜6日は入りたいです。ここメインにしたいので
```

**確認観点**
- 経験者ルートで score が高く（80〜100）計算されるか
- Grade A になるか
- 時給・指名料の質問に対して適切な返答が出るか

## テスト観点チェックリスト

### 問診フロー
- [ ] 経験者ルートと未経験ルートが正しく分岐するか
- [ ] すべてのペルソナで問診が最後まで到達できるか
- [ ] 途中で止まる・繰り返す・スキップされる質問がないか

### スコアリング
- [ ] P03・P08 が Grade A になるか
- [ ] P01・P05 が Grade B 前後になるか
- [ ] P06・P10 が Grade C 前後になるか
- [ ] P02・P07 が Grade D になるか

### Intent 分類
- [ ] 「身バレ・バレる」が適切に分類されるか（P01・P04）
- [ ] 「学校・テスト・大学」が other になっていないか（P07）
- [ ] 「子供・育児・急な欠席」が other になっていないか（P10）
- [ ] 「ガールズバー・スナック・経験の種類」が正しく扱われるか（P09）
- [ ] 「掛け持ち・Wワーク」が正しく扱われるか（P04・P07）

### LINE CTA
- [ ] 問診完了後に LINE CTA が表示されるか
- [ ] P02・P06・P07 でも CTA が表示されるが「押さない」選択ができるか

### 管理画面
- [ ] テスト実施後に crew_applicants に正しいデータが保存されるか
- [ ] score_detail に経験者/未経験ルートが正しく記録されるか
- [ ] 管理画面「応募者一覧」にテストデータが反映されるか
